## 阶段1: 策划与定义

### 目标
定义项目范围，建立项目基础，获取跨职能团队承诺。

### 关键输入
- 顾客Voice (VOC) / 市场需求
- 业务计划/营销策略
- 产品/过程假设
- 顾客要求(技术规范、特殊特性)

### 关键活动

| 活动 | 说明 | 责任人 |
|------|------|--------|
| 项目立项 | 正式批准启动APQP项目 | 项目经理 |
| 多方论证会议 | 跨职能团队评审项目可行性 | 项目经理 |
| 确定项目范围 | 定义边界、排除项 | 团队 |
| 建立项目团队 | 确定成员与职责分工 | 项目经理 |
| 初始BOM建立 | 物料清单初版 | 采购/工程 |
| 确定目标成本 | 基于市场与战略 | 财务 |
| 制定项目计划 | 里程碑与交付物 | 项目经理 |
| 识别顾客特殊要求 | SC/CC识别 | 质量 |

### 阶段输出清单

- [ ] 项目章程 (Project Charter)
- [ ] 多方论证会议记录
- [ ] 初始过程流程图
- [ ] 产品特殊性初步清单
- [ ] 项目团队名单与RACI
- [ ] 初始BOM
- [ ] 项目进度计划(里程碑)
- [ ] 成本目标

### 节点检查表

| 检查项 | 完成标准 |
|--------|----------|
| 项目授权 | 管理层签署批准 |
| 团队组建 | 所有核心成员已确认 |
| 范围定义 | 边界图完成 |
| 风险评估 | 初步FMEA启动 |
| 顾客沟通 | 特殊要求已确认 |

## 阶段2: 产品设计开发

### 目标
设计满足顾客要求的产品，验证设计能力。

### 关键输入
- 阶段1输出
- 顾客设计要求
- 法规要求
- 类似产品经验

### 关键活动

| 活动 | 说明 | 责任人 |
|------|------|--------|
| 产品设计 | DFMEA启动 | 设计工程师 |
| 设计评审DR1 | 概念设计评审 | 团队 |
| 设计评审DR2 | 详细设计评审 | 团队 |
| 设计评审DR3 | 设计验证评审 | 团队 |
| DFMEA维护 | 持续更新 | 设计工程师 |
| DVP&R执行 | 设计验证计划与报告 | 测试工程师 |
| 设计变更管理 | EC执行与跟踪 | 工程 |
| 样件制作 | 首批样件 | 工艺/制造 |

### 阶段输出清单

- [ ] DFMEA (设计FMEA)
- [ ] 设计评审记录 (DR1/DR2/DR3)
- [ ] 设计验证报告 (DVP&R)
- [ ] 特殊特性清单(设计)
- [ ] 图纸与规范
- [ ] 样件
- [ ] 设计变更记录

### DFMEA填写要点

| 要素 | 填写指导 |
|------|----------|
| 功能 | 描述设计意图与功能 |
| 潜在失效模式 | 可能发生的失效方式 |
| 潜在失效影响 | 失效对顾客的影响 |
| 严重度S | 1-10评分，参考评分准则 |
| 潜在原因/机理 | 导致失效的原因 |
| 频度O | 1-10评分，基于类似设计历史 |
| 现行设计控制-预防 | 防止原因发生的控制 |
| 现行设计控制-探测 | 探测原因的控制 |
| 探测度D | 1-10评分 |

### 节点检查表

| 检查项 | 完成标准 |
|--------|----------|
| DFMEA完整性 | 所有功能已分析 |
| 设计验证通过 | DVP&R全部项目PASS |
| 样件认可 | 符合设计意图 |
| 特殊特性确认 | 顾客已批准 |

## 阶段3: 过程设计开发

### 目标
开发满足产品要求的制造过程。

### 关键输入
- 阶段2输出
- 批准的设计
- 样件认可结果
- 设备/工装信息

### 关键活动

| 活动 | 说明 | 责任人 |
|------|------|--------|
| 过程流程图 | PFMEA启动 | 工艺工程师 |
| PFMEA编制 | 过程风险分析 | 工艺/质量 |
| 控制计划编制 | 量产控制策略 | 质量工程师 |
| 作业指导书 | 标准操作程序 | 工艺工程师 |
| 工装夹具设计 | 检具/量具开发 | 工艺 |
| 测量系统分析 | MSA计划与执行 | 质量 |
| 过程能力研究 | Ppk/Cpk评估 | 质量 |
| 产能规划 | 平衡分析与优化 | 制造 |

### 阶段输出清单

- [ ] 过程流程图
- [ ] PFMEA (过程FMEA)
- [ ] 控制计划
- [ ] 作业指导书(SOP)
- [ ] 检具/量具清单
- [ ] MSA报告
- [ ] 初始过程能力报告
- [ ] 产能规划

### PFMEA与DFMEA区别

| 项目 | DFMEA | PFMEA |
|------|-------|-------|
| 分析对象 | 产品设计 | 制造过程 |
| 关注点 | 设计缺陷 | 过程变异 |
| 失效模式 | 功能失效 | 缺陷/不良 |
| 现行控制 | 设计控制 | 过程控制 |

### 节点检查表

| 检查项 | 完成标准 |
|--------|----------|
| PFMEA完整性 | 所有工序已分析 |
| 控制计划批准 | 已获质量/工程签字 |
| 过程能力达标 | Ppk >= 1.33 (关键特性) |
| 作业指导书培训 | 操作员100%培训 |
